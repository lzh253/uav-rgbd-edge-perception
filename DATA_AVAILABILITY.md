# Data Availability

The experiments use the publicly available AeroScapes dataset. The dataset is not redistributed in this repository.

This repository provides:

- fixed train, validation, and test splits;
- training and evaluation scripts;
- portable configuration files;
- compact aggregate result tables; and
- the validation-selected latent-8 checkpoint through the `v1.0.0` release.

Generated depth arrays, per-sample predictions, and raw dataset files must be created locally.
